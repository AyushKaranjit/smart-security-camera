# Initial Build Plan

## Core features
1. Load video from file
2. Preprocess frames
3. Detect motion
4. Define unauthorized-access region (ROI)
5. Save annotated incident clips
6. Save incident metadata/log

## Extra features
1. Object tracking
2. Human/vehicle categorisation
3. Streamlit dashboard for review

## Evaluation targets
- At least 10 test cases
- Incident examples saved in outputs/
- Screenshots and figures for report

## Constraints
- Python only
- Simple, defendable implementation
- Must be explainable in viva
